# 사다리타기 앱 - 당근을 찾아라

인원 수만큼 사다리를 생성해 순서대로 내려간 결과로 당첨자를 결정하는 Android 앱입니다.

---

## 주요 기능

- 인원 수 입력 시 자동으로 사다리 라인 생성 (최대 10명)
- 한 명씩 순서대로 사다리를 타며 결과 확인
- 마지막 인원까지 완료되면 당첨자 공개
- 당첨은 '당근'으로 나머지는 '토끼'로 표시됨
